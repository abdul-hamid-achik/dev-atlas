## Installation

### From VSIX (Recommended)

1. Build the extension:
```bash
cd apps/vscode-extension
npm run build
npm run package
```

2. Install in VS Code/Cursor:
   - Open Extensions view (Ctrl+Shift+X)
   - Click "..." menu â†’ "Install from VSIX..."
   - Select the generated `.vsix` file

### From Source

1. Clone and build:
```bash
git clone https://github.com/username/dev-atlas.git
cd dev-atlas/apps/vscode-extension
npm install
npm run build
```

2. Run in development:
   - Press F5 to open Extension Development Host
   - Test your changes in the new VS Code window

## Commands

The extension provides several commands accessible through the Command Palette (Ctrl+Shift+P):

### `Dev Atlas: Open Knowledge Graph`
Opens the main knowledge graph view in the sidebar.

### `Dev Atlas: Create Node`
Creates a new node in the knowledge graph:
1. Enter node name
2. Select node type (Component, Service, Concept, etc.)
3. Add optional description

### `Dev Atlas: Create Edge`
Creates a relationship between two nodes:
1. Select source node from dropdown
2. Select target node from dropdown  
3. Define relationship type (depends_on, implements, uses, etc.)

### `Dev Atlas: Search Nodes`
Search for nodes in the knowledge graph:
- Full-text search across node names and observations
- Filter by node type
- Sort by relevance or creation date

### `Dev Atlas: Export Graph`
Export the knowledge graph in various formats:
- JSON format for backup/restore
- GraphML for external visualization tools
- CSV for spreadsheet analysis

## Configuration

### Extension Settings

Configure the extension through VS Code settings:

```json
{
  "dev-atlas.server.host": "localhost",
  "dev-atlas.server.port": 3000,
  "dev-atlas.ui.theme": "auto",
  "dev-atlas.ui.compactMode": false,
  "dev-atlas.search.fuzzyMatching": true,
  "dev-atlas.graph.maxNodes": 1000
}
```

### Keybindings

Default keyboard shortcuts:

- `Ctrl+Shift+K G`: Open Knowledge Graph
- `Ctrl+Shift+K N`: Create Node  
- `Ctrl+Shift+K E`: Create Edge
- `Ctrl+Shift+K S`: Search Nodes

Customize in VS Code's Keyboard Shortcuts (Ctrl+K Ctrl+S).

## Workspace Integration

### File Associations

- Drag files onto nodes to create associations
- Auto-detect relevant files based on node type
- Navigate from graph to code and vice versa

### Git Integration

## Troubleshooting

### Common Issues

**Extension not loading:**
- Check VS Code version compatibility (>= 1.74.0)
- Verify extension is enabled
- Check Output panel for error messages

**Cannot connect to MCP server:**
- Ensure MCP server is running
- Verify host/port configuration
- Check firewall settings

**Graph not updating:**
- Try refreshing the view (Ctrl+R)
- Check MCP server logs
- Verify database permissions

### Debug Mode

Enable debug logging:

```json
{
  "dev-atlas.debug.enabled": true,
  "dev-atlas.debug.logLevel": "verbose"
}
```

View logs in Output panel â†’ "Dev Atlas" channel.
