# Dev Atlas 🗺️

**Mapea tu conocimiento de desarrollo como nunca antes**

Dev Atlas es una solución completa para crear y gestionar knowledge graphs específicamente diseñados para desarrolladores. Combina un potente servidor MCP (Model Context Protocol) con una extensión elegante para VS Code/Cursor.

## ¿Qué es Dev Atlas?

Dev Atlas transforma la manera en que organizas y navegas el conocimiento de desarrollo. No más información dispersa en notas, documentos o comentarios perdidos en el código.

<Callout type="info" emoji="💡">
  **Knowledge Graph**: Una representación visual y estructurada de conceptos, relaciones y dependencias en tu ecosistema de desarrollo.
</Callout>

## ¿Por qué Dev Atlas?

### 🧠 **Externaliza tu Memoria de Desarrollo**
Deja de confiar solo en tu memoria. Captura patrones, decisiones arquitecturales, y relaciones entre componentes de manera visual.

### 🔗 **Visualiza Dependencies**
Entiende cómo se conectan tus bibliotecas, módulos y conceptos en un grafo interactivo.

### 🚀 **Acelera Onboarding**
Nuevos miembros del equipo pueden entender la arquitectura y relaciones del código más rápidamente.

### 📚 **Documenta Dinámicamente** 
Tu knowledge graph evoluciona con tu código, manteniendo la documentación siempre actualizada.

## Comenzar es Fácil

```bash
# Clona el repositorio
git clone https://github.com/username/dev-atlas.git
cd dev-atlas

# Instala dependencias
npm install

# Ejecuta el servidor MCP
npm run dev --workspace=@dev-atlas/mcp-knowledge-graph

# En otra terminal, desarrolla la extensión
npm run dev --workspace=@dev-atlas/vscode-extension
```
